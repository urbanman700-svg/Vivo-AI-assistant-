# 🔤 Font Setup Guide — Vivo Assistant

## Required Fonts

The app uses two font families:

| Font | Usage | Download |
|------|-------|----------|
| **Orbitron Bold** | App name, headings | Google Fonts |
| **Inter Regular** | Body, descriptions | Google Fonts |
| **Inter SemiBold** | Subheadings, labels | Google Fonts |

---

## Step-by-Step Setup in Android Studio

### Option A — Download Fonts Manually

1. Go to [Google Fonts](https://fonts.google.com)
2. Download **Orbitron** (select Bold 700)
3. Download **Inter** (select Regular 400 + SemiBold 600)

4. Place `.ttf` files in:
```
app/src/main/res/font/
├── orbitron_bold.ttf
├── inter_regular.ttf
└── inter_semibold.ttf
```

### Option B — Use Android Studio Font Downloader (Recommended)

1. Open any XML layout file
2. Click on a `android:fontFamily` attribute
3. Select **More Fonts...**
4. Search **Orbitron** → Select Bold → Click **OK** → Choose **Add font to project**
5. Repeat for **Inter** (Regular + SemiBold)

Studio will auto-create the font folder + XML descriptors.

---

## Font XML Descriptors (auto-generated by Studio)

After downloading, Android Studio creates these in `res/font/`:

**orbitron_bold.xml**
```xml
<?xml version="1.0" encoding="utf-8"?>
<font-family xmlns:app="http://schemas.android.com/apk/res-auto">
    <font
        app:fontStyle="normal"
        app:fontWeight="700"
        app:font="@font/orbitron_bold" />
</font-family>
```

**inter_regular.xml**
```xml
<?xml version="1.0" encoding="utf-8"?>
<font-family xmlns:app="http://schemas.android.com/apk/res-auto">
    <font
        app:fontStyle="normal"
        app:fontWeight="400"
        app:font="@font/inter_regular" />
    <font
        app:fontStyle="normal"
        app:fontWeight="600"
        app:font="@font/inter_semibold" />
</font-family>
```

---

## Fallback (No Font Setup)

If you skip font setup, replace all font references in XML layouts:

```
android:fontFamily="@font/orbitron_bold"   →  android:fontFamily="sans-serif-condensed"
android:fontFamily="@font/inter_semibold"  →  android:fontFamily="sans-serif-medium"
android:fontFamily="@font/inter_regular"   →  android:fontFamily="sans-serif"
```

The app will still work — just without the futuristic Orbitron typeface.
